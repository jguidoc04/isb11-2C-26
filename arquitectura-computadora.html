Nivel 01 · Base física
Transistores
Un transistor es un interruptor electrónico microscópico hecho de silicio. Puede estar encendido (1) o apagado (0). Los procesadores modernos contienen miles de millones de ellos.
⚡
Efecto de campo (FET)
Una tensión en la compuerta controla el flujo de corriente entre fuente y drenaje.
🔲
MOSFET
El tipo más común hoy. Extremadamente pequeño — actualmente ~3-4 nm en producción.
🔢
Binario
0 V ≈ "0", 1.8–3.3 V ≈ "1". Todo el cómputo se reduce a este hecho.

Compuertas Lógicas

Universalidad NAND
Con solo compuertas NAND se puede construir cualquier otra función lógica.

Las leyes de De Morgan y la distribución permiten simplificar circuitos.
⏱️
Retardo de propagación
Cada compuerta tiene un retardo mínimo (~ps). Limita la velocidad del reloj.

Full Adder

MUX como bus

Nivel 04 · Cálculo
Unidad Aritmético-Lógica (ALU)
La ALU ejecuta todas las operaciones aritméticas (suma, resta) y lógicas (AND, OR) sobre los datos. Es el "motor de cálculo" del procesador.
// ALU de 8 bits — selecciona operación e introduce valores

🚦
Flags de estado
Zero (Z), Carry (C), Overflow (V), Negative (N) — el resultado afecta los bits de condición.
↔️
Complemento a dos
La resta se implementa sumando el complemento a dos del sustraendo.
⚙️
Opcode
La señal de control selecciona qué operación realiza la ALU via un MUX interno.
Nivel 05 · Almacenamiento rápido
Registros
Los registros son celdas de memoria ultra-rápidas dentro del procesador. Construidas con flip-flops, guardan operandos y resultados temporales. Acceso en un solo ciclo de reloj.
// Banco de registros de 8 bits (haz clic para escribir un valor)
⚡
Velocidad
~1 ciclo (< 1ns). Los datos en registros no viajan al bus de memoria.
📌
PC y SP
Program Counter y Stack Pointer son registros especiales con funciones fijas en la arquitectura.
🔒
Flip-Flop D
Un bit de registro es un flip-flop D: captura el dato en el flanco del reloj y lo retiene.
Nivel 06 · Almacenamiento
Memoria
La memoria almacena instrucciones y datos. Existen múltiples niveles con diferente velocidad y capacidad: caché L1/L2/L3 dentro del chip, RAM principal, y almacenamiento masivo.

🎯
Localidad de referencia
Los programas tienden a acceder a datos cercanos en el tiempo y el espacio — base del caché.
📍
Dirección de memoria
Cada byte tiene una dirección única. En 64 bits hay hasta 2⁶⁴ ≈ 18 quintillones de bytes.
Nivel 07 · Dirección
Unidad de Control
La UC lee instrucciones de memoria, las decodifica y genera las señales de control que coordinan ALU, registros y memoria. Opera en ciclos repetitivos: Fetch → Decode → Execute.

Guardar resultado
El Program Counter (PC) apunta a la siguiente instrucción. La UC la copia en el Instruction Register (IR) y avanza el PC.
⏰
Reloj (clock)
Sincroniza todo. A 3 GHz hay 3×10⁹ ciclos por segundo. Cada instrucción puede tomar 1-20+ ciclos.
🔄
Pipeline
Superpone las fases de varias instrucciones para aumentar el throughput sin subir la frecuencia.
📝
ISA
La Instruction Set Architecture define el "idioma" que entiende la UC: x86, ARM, RISC-V…
Nivel 08 · Procesador
CPU
La CPU integra la ALU, los registros y la Unidad de Control en un solo chip. Un bus de datos interno conecta todos los componentes y los sincroniza con el reloj del sistema.

🧠
Superescalaridad
Los CPUs modernos ejecutan múltiples instrucciones por ciclo usando varias ALUs en paralelo.
🔮
Predicción de saltos
La CPU predice el resultado de condicionales para no detener el pipeline.
🌡️
Disipación de calor
Miles de millones de transistores conmutando generan calor. El TDP define la disipación de un chip.
Nivel 09 · Sistema completo
Computadora
Una computadora es el sistema completo: CPU + Memoria principal + Almacenamiento + Entrada/Salida + Software. Todo construido desde el transistor, capa a capa.
// Capas de abstracción completas
🖥️
Von Neumann
Arquitectura que unifica datos e instrucciones en la misma memoria. Base de casi todas las computadoras modernas.
🔌
Buses del sistema
Address bus, Data bus y Control bus conectan CPU, memoria y periféricos en un canal compartido.
💾
Sistema Operativo
El SO abstrae el hardware y arbitra los recursos (CPU, memoria, I/O) entre los procesos.
📱
Aplicaciones
El software de alto nivel se compila hasta instrucciones de máquina que la UC ejecuta ciclo a ciclo.
